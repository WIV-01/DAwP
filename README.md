# 📊 CO2, Energie en GDP Analyse

Winc Academy project – Data Analyst with Python

---

## 📌 Projectbeschrijving
Analyse van wereldwijde data over:

- CO2-uitstoot  
- Energieverbruik  
- GDP per capita  

De data wordt opgehaald via *Our World in Data* en geanalyseerd met Python.

---

## ❓ Onderzoeksvragen
1. Wat is de relatie tussen CO2-uitstoot, energieverbruik en GDP?  
2. Welke landen verminderen hun CO2-uitstoot het meest?  
3. Hoe is het aandeel van verschillende energiebronnen verdeeld?  

---

## 🛠️ Gebruikte technologieën
- Python  
- Pandas  
- NumPy  
- Matplotlib
